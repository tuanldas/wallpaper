# Wallpaper Templates

## Description
The built-in wallpaper template library in the app, organized into 11 categories. Users can browse, preview, and set them as wallpaper.

## Access
Home > Category tabs > Tap on a wallpaper

## Categories (11 types)
1. **For you** - Suggestions based on user interests
2. **DIY** - Scrapbook/collage style, polaroid photos
3. **Love** - Love theme: heart headphones, bunny couple, heart candy
4. **Live** - Live wallpapers with animated effects: anime scenes, flowers, nature
5. **Lock Screen** - Wallpapers for lock screen
6. **Kawaii** - Cute Japanese style theme
7. **Aesthetic** - Aesthetic theme, pastel
8. **Quote** - Wallpapers with quotes
9. **Famous** - Celebrities
10. **Car** - Car theme
11. **Anime** - Anime/manga theme

## Wallpaper Types
- **Static wallpapers**: Still images
- **Live wallpapers**: Animated effects, marked with "(>) Live" badge

## Template Viewing Flow
1. Select a category tab
2. Scroll through wallpaper grid (3 columns, infinite scroll)
3. Tap wallpaper -> Video ad -> Preview screen
4. Preview: swipe left/right to browse
5. "Next" -> Set wallpaper screen
6. "Set wallpaper" -> Set as phone wallpaper
