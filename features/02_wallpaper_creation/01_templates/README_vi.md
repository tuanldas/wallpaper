# Wallpaper Templates

## Mô tả
Thư viện wallpaper templates có sẵn trong app, được phân loại theo 11 categories. Người dùng có thể duyệt, xem preview và set làm wallpaper.

## Vị trí truy cập
Home > Category tabs > Tap vào wallpaper

## Categories (11 loại)
1. **For you** - Gợi ý theo sở thích người dùng
2. **DIY** - Kiểu scrapbook/collage, polaroid photos
3. **Love** - Theme tình yêu: headphone trái tim, bunny couple, kẹo trái tim
4. **Live** - Live wallpapers với hiệu ứng động: anime scenes, hoa, thiên nhiên
5. **Lock Screen** - Wallpaper cho màn hình khóa
6. **Kawaii** - Theme dễ thương kiểu Nhật Bản
7. **Aesthetic** - Theme thẩm mỹ, pastel
8. **Quote** - Wallpaper có câu nói/quotes
9. **Famous** - Người nổi tiếng
10. **Car** - Theme xe hơi
11. **Anime** - Theme anime/manga

## Loại wallpaper
- **Static wallpapers**: Hình ảnh tĩnh
- **Live wallpapers**: Có hiệu ứng động (animation), đánh dấu bằng badge "(>) Live"

## Flow xem template
1. Chọn category tab
2. Scroll grid wallpapers (3 cột, infinite scroll)
3. Tap wallpaper -> Quảng cáo video -> Preview screen
4. Preview: swipe trái/phải để duyệt
5. "Next" -> Set wallpaper screen
6. "Set wallpaper" -> Đặt làm wallpaper điện thoại
